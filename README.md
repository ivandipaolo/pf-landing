## Getting Started

### Installation and usage

#### For development:


1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/ivandipaolo/pf-landing.git
   ```
   
2. Install dependencies

   ```bash
   npm install
   ```

3. Run dev server
(Default port 3000)
   ```bash
   npm run dev
   ```

4. Open and start developing

   ```bash
   localhost:3000
   ```

#### For deployment:

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/ivandipaolo/pf-landing.git
   ```
   
2. Install dependencies

   ```bash
   npm install
   ```

3. Create build

   ```bash
   npm run build
   ```

4. Run preview server
(Default port 4173)
   ```bash
   npm run preview
   ```

5. Open preview

   ```bash
   localhost:4173
   ```
